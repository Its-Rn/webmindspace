# Personal Productivity Platform

This repository starts with a Phase 1 production scaffold for a full-stack personal productivity platform.

## Stack

- Frontend: React, Vite, Tailwind CSS, React Router, Axios, Framer Motion, React Hook Form, React Icons, React Query
- Backend: Node.js, Express.js
- Database: MongoDB Atlas with Mongoose
- Auth and media: JWT, refresh tokens, bcrypt, email verification, Multer, Cloudinary, Socket.IO

## Current Phase

- Phase 5: Dashboard

## Structure

- `client/`: React application
- `server/`: Express API
- `server/src/config`: runtime and environment configuration
- `server/src/controllers`, `server/src/middleware`, `server/src/models`, `server/src/routes`, `server/src/services`, `server/src/validators`, `server/src/utils`: scalable backend modules
- `server/uploads` and `server/logs`: runtime storage

## Getting Started

1. Copy `.env.example` to `.env` files as needed.
2. Run `npm install` from the repository root.
3. Start both apps with `npm run dev`.
